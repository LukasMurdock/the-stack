# Build

Use this track when shipping product features in your first month.

## Start here

1. Follow the 10-minute quickstart in `README.md`.
2. Configure auth mode (`AUTH_SIGNUP_MODE`) for your environment.
3. Run local migrations and seed data.
4. Verify `/api/health` and `/api/scalar`.
