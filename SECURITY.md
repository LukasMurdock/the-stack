# Security Policy

## Reporting a Vulnerability

Please report security issues through this GitHub repository.

Preferred: open a private report through GitHub Security Advisories.

If private reporting is not available, open an issue in this repository and avoid sharing exploit details.

## Response Expectations

- We aim to acknowledge new reports within 72 hours.
- We will triage severity and provide status updates in the GitHub thread.
- We will coordinate disclosure timing for confirmed vulnerabilities.

## Scope

This policy applies to this repository and its default branch.
